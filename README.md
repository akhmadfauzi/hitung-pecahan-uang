# Run

Hitung Pecahan Uang is a simple application for displaying denominations based from users input.

## Installation

Use the node package manager to install this application.

```bash
npm install
```

## Usage
To start development mode.
```bash
npm start
```

Build production mode.
```bash
npm run build
```

Test.
```bash
npm run test
```
